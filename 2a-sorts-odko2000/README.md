# 3 sorts
`sort.c` файл дотор insertion, selection, bubble эрэмбэлэх аргуудыг хэрэгжүүл.

  * Оруулан эрэмбэлэх
    ```C
    void insertion_sort(int a[], int n)
    ```
    `a`: эрэмбэлэх хүснэгт  
    `n`: хүснэгтэд хадгалагдсан утгын тоо  
    
  * Сонгон эрэмбэлэх
    ```C
    void selection_sort(int a[], int n)
    ```
    `a`: эрэмбэлэх хүснэгт  
    `n`: хүснэгтэд хадгалагдсан утгын тоо  
    
  * Хөөсөн эрэмбэлэлт
    ```C
    void bubble_sort(int a[], int n)
    ```
    `a`: эрэмбэлэх хүснэгт  
    `n`: хүснэгтэд хадгалагдсан утгын тоо
